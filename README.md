## Overview
GUI-based Python math game designed to help younger students practice fundamentals with interactive flashcards.

## Features
- Difficulty + operator selection
- Optional negatives
- Configurable number of questions
- In-game UI controls (Back/Start/Enter/Reset/Quit)
- Correct/wrong/total counters, progress bar, stopwatch

## Files
- `MathGalaxy.py` (main)
- `GUIscreen.py` (UI)
- `Images/` (assets)
- `README.md`

## Getting started
1. Download the repo
2. Move all images to the main directory
3. Install python-tk (Ensure minimum 8.6)
4. Run python MathGalaxy.py (from the Math-Galaxy folder)
